LIR – "Lehr- und Lernsystem Information Retrieval"

LIR is an experimental retrieval environment to teach/learn information retrieval concepts. It's designed to be highly configurable and parametrisable. Refer to the original project overview and documentation for more background information.

  • lir.pl – The retrieval front-end
  • lir_index.pl (restricted access) – The web front-end for creating indexes for lir.pl
